# 双语论文 PDF 出版式排版设计

## 目标

统一修复 PaperLab 的 PDF 出版质量。中文全文对照版采用 A 方案：英文原文作为紧凑校勘层，中文译文作为主要阅读层；精读版沿用同一字体、页边距、标题、公式、图表和参考文献规则。

## 版式

- A4，正文区采用出版物常见的窄行宽和上下留白。
- 中文正文使用宋体风格字体，英文原文使用 Georgia；标题使用黑体风格字体。
- 每个双语单元显示为浅灰英文原文块，随后是带暗红左线的中文译文块。
- `原文/译文 + 单元 ID` 只作为小号标签，不再显示成重复的大标题。
- 公式居中并保留编号；图表不拆页；参考文献使用普通衬线正文，不显示成代码灰框。
- 页眉显示文稿标题与版本类型；页脚显示页码。
- 不生成“人工确认”区域。

## 实现边界

只修改共享导出路径 `.paperlab/paperlab.py`，不为十份文稿分别编写版式。先用 AlexNet 短稿验收，再统一替换十份正式 PDF 并同步 Obsidian。

## 验收

- HTML 中双语单元具有稳定语义类名，顺序仍为原文后译文。
- 普通金额不会被当作公式；数字开头的合法公式不会吞掉相邻中文。
- PDF 中没有原始 LaTeX、乱码、“人工确认”或空白页。
- 抽查标题层级、双语块、公式、图表、参考文献、页眉和页码均可读。
